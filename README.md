# api-nodejs

> API em node para requisições em banco de dados MongoDB.

# New Features!

  - Save new user in mongoDB.

### Require
| Requeriment | URL |
| ------ | ------ |
| MongoDB | https://www.mongodb.com/ |
| NodeJS | https://nodejs.org/ |

### Installation
Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ node src/index.js
```
